testApp - Test application for GCE
====

Overview

This application is a sample running on GCE.
When requesting an endpoint of `test 3`, read and write random data to memcached on the network.

## Requirement
  - You will need a bundler.

## Usage
  - `bundle install`
  - `bash ./start.sh`
